# 📈 SQL for Business Intelligence: A Case Study Portfolio

Welcome! This repository serves as a professional portfolio of my work in SQL for data analysis and business intelligence. Here, I tackle real-world business problems by transforming raw data into actionable insights.

## 📂 Case Studies Overview

Here is a summary of the projects you'll find in this repository. Each folder contains the problem statement, my complete SQL script, and a summary of the key findings.

| Case Study Name          | Business Problem                                      | Key SQL Concepts Used                                    |
| ------------------------ | ----------------------------------------------------- | -------------------------------------------------------- |
| **Customer Segmentation**| Grouping customers into distinct segments for targeted marketing. | `CASE` Statements, `CTEs`, `Joins`, `Aggregate Functions`  |
| **Sales Performance Analysis** | Identifying top-performing products and sales trends over time.  | `Window Functions` (`RANK`, `LAG`), `Date Functions`, `Grouping` |
| **Product Funnel Analysis** | Analyzing user drop-off rates at different stages of a process. | `Self-Joins`, `Subqueries`, `Conditional Aggregation`      |
